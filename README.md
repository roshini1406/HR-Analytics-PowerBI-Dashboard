📊 HR Analytics Dashboard – Power BI Project

📌 Project Overview

This project showcases an HR Analytics Dashboard built using Microsoft Power BI.  
The dashboard analyzes employee data to uncover insights about workforce trends, employee attrition, salary distribution, and demographics.

The goal of this project is to transform raw HR data into meaningful visual insights that support data-driven decision making.



🛠 Tools & Technologies
- Microsoft Power BI
- Power Query
- Data Visualization Techniques



📂 Dataset Information
The dataset includes employee-related attributes such as:

- Age
- Gender
- Department
- Job Role
- Monthly Income
- Attrition
- Overtime
- Education Field
- Years at Company



 📈 Dashboard Features

🔹 KPI Metrics
- Total Employees
- Average Age
- Average Salary
- Attrition Count

 📊 Visualizations
1. Attrition Distribution
2. Department-wise Employee Count
3. Attrition by Department
4. Gender Distribution
5. Salary by Job Role
6. Overtime vs Attrition
7. Age Distribution
8. Education Field Analysis


🔍 Key Insights
- Certain departments show higher employee attrition.
- Overtime may influence employee attrition.
- Salary levels vary across different job roles.
- Workforce demographics provide insights into employee distribution.
